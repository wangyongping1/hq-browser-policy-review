## hq-browser 说明

`hq-browser` 是对 `agent-browser` 的包装。

它主要做：

- 自动读取正确的 CDP URL。
- 自动指定 Chrome engine。
- 对同一个 CDP endpoint 加锁，避免并发抢同一个 relay/bridge 会话。
- 提供 `reset`，清理锁并执行 `agent-browser close --all`，避免复用旧 session。
- 将常规命令转发给底层 `agent-browser`。

常用命令：

```bash
hq-browser reset
hq-browser open https://example.com
hq-browser get url
hq-browser --json snapshot
hq-browser click @e4
hq-browser wait 1500
hq-browser eval --stdin < script.js
hq-browser close --all
```

重要约束：

- 控制用户本机浏览器时只使用 `hq-browser`。
- 不直接调用 `agent-browser`。
- 不使用 `ws://lightpanda:9223`。
- 不切到 Camofox、Lightpanda 或默认 browser tool 作为替代。
